Expand description

A command to be executed, provided in command-flags-keywords form.

Fields

sudo: bool

Flag indicating If a normal admin needs to run this command with sudo.

cmd: Vec<String>

The “command” part of the command string, eg. brew install.

flags: Vec<String>

The “flags” part of the command string, eg. --dry-run.

kws: Vec<String>

The “keywords” part of the command string, eg. curl fish.

async fn exec_prompt(self, mute: bool) -> Result<Vec<u8>, Error>

Executes a Cmd and collects its stderr.

If mute is false, then its stderr output will be printed to stderr too.

This function behaves just like exec_checkerr, but in addition, the user will be prompted if (s)he wishes to continue with the command execution.
