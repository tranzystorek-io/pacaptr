pub struct Pacaptr {
    ops: Operations,
    using: Option<String>,
    dry_run: bool,
    needed: bool,
    no_confirm: bool,
    no_cache: bool,
    keywords: Vec<String>,
    extra_flags: Vec<String>,
}
Expand description

The command line options to be collected.

Fields

ops: Operationsusing: Option<String>

Specify the package manager to be invoked.

dry_run: bool

Perform a dry run.

needed: bool

Prevent reinstalling previously installed packages.

no_confirm: bool

Answer yes to every question.

no_cache: bool

Remove cache after installation.

keywords: Vec<String>

Package name or (sometimes) regex.

extra_flags: Vec<String>

Extra Flags passed directly to backend.
